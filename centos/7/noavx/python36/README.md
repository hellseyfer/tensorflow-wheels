# Built for CentOS/RHEL 7 on Westmere with no AVX and no GPU

## Build Versions
* TensorFlow 1.12.2 
* Python 3.6.3
* Bazel 0.15.2
* GCC 4.8

----

## Install Instructions

```
pip install /path/to/tensorflow-1.12.2-cp36-cp36m-linux_x86_64.whl
```
